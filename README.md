# 함께 만드는 숲 이야기

`함께 만드는 숲 이야기`는 비사이드에서 주관한 `포텐데이`에 참여하여 진행한 프로젝트입니다.

10일 동안 아이템을 선정하여 디자인하여 프로토 타입을 구현하는 행사입니다.

## 개발 환경

- Intellij IDEA Ultimate 2023.1.3
- Java 17
- Gradle 7.6.1
- Spring Boot 2.7.6


## 기술 스택

- Spring Web
- Spring Data JPA
- Spring Security
- Spring Data Redis
- Spring OAuth2 Client
- Spring Configuration Processor
- Jackson
- Base62
- JWT
- Lombok
- MySQL
- H2
